## Ozon Calculus LaTeX Shortcuts

| Shortcut | Description | LaTeX Output |
|---------|-------------|--------------|
| `int` | Indefinite integral with dx | `\int \square\,\mathrm{d}x` |
| `dint` | Definite integral | `\int_{\square}^{\square}\square\,\mathrm{d}x` |
| `ddx` | Derivative w.r.t. x | `\frac{d}{dx}\square` |
| `ddy` | Derivative w.r.t. y | `\frac{d}{dy}\square` |
| `d2dx2` | Second derivative | `\frac{d^{2}}{dx^{2}}\square` |
| `dndxn` | nth derivative | `\frac{d^{\square}}{dx^{\square}}\square` |
| `deriv` | Derivative at a point | `\frac{d}{d\square}\square\bigm|` |
| `pdx` | Partial derivative (x) | `\frac{\partial}{\partial x}\square` |
| `pdy` | Partial derivative (y) | `\frac{\partial}{\partial y}\square` |
| `lim` | General limit | `\lim_{\square\to \square}\square` |
| `limx` | Limit with x | `\lim_{x\to \square}\square` |
| `sum` | General summation | `\sum_{\square}^{\square}\square` |
| `sumn` | Summation with n | `\sum_{n=\square}^{\square}\square` |
